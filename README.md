# collision-probability

#### Usage

Compile

    make collision_probability

Produce data file

    make samples.dat

Plot results

    make plot

#### Dependencies

- [Gnuplot](http://www.gnuplot.info)
- [Eigen](http://eigen.tuxfamily.org/)

Modify `Makefile` to fix include directory for Eigen and library linking path.
